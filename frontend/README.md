# 信访智能文书生成系统 - 前端

## 技术栈

- Vue 3
- TypeScript
- Vite
- Element Plus
- Pinia
- ProseMirror

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

访问 http://localhost:5173

## 构建

```bash
npm run build
```

## 主要功能

1. **用户认证**
   - 登录、注册
   - JWT 令牌管理

2. **文件管理**
   - 文件上传
   - 在线预览
   - 文件列表

3. **文书研判**
   - AI 智能研判
   - 错误标注
   - 在线修正

4. **文书生成**
   - AI 对话式生成
   - 模板选择
   - 实时预览

5. **模板管理**
   - 模板创建
   - 模板列表
   - 模板使用

6. **版本管理**
   - 修改记录
   - 版本对比
   - 精细化回滚
