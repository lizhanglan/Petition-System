## 🔧 快速命令

### 启动服务
```bash
# 后端
cd backend
python run.py

# 前端
cd frontend
npm run dev
```

### 安装依赖
```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd frontend
npm install
```

### 测试 API
```bash
# 健康检查
curl http://localhost:8000/

# 版本列表
curl -H "Authorization: Bearer TOKEN" \
  http://localhost:8000/api/v1/versions/list/1
```
